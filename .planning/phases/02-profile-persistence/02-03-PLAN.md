5. Delete profiles\TestProfile.json

2. Test sprint/dodge: hold the Dodge Roll key — confirm output key fires; release — confirm it stops (HoldToToggle for SprintKey; EdgeTrigger for DodgeKey)

4. Switch to Toxic Commando
5. Test scroll-wheel mapping: press ScrollUpKey — should scroll up; press ScrollDownKey — should scroll down

**Scenario 4 — PROF-04: bindings load from JSON (settings.ini migration)**
1. Exit the app
2. Delete or rename settings.ini so it does not exist
3. Launch the app, select Elden Ring, open settings — all bindings should show their JSON defaultVk values (F, C, CapsLock, Escape)
4. Change one binding (e.g., change DodgeKey from C to V)
5. Click Save
6. Exit the app and relaunch, select Elden Ring, open settings — DodgeKey should still show V (persisted to JSON)
7. Open profiles\EldenRing.json in a text editor — confirm the DodgeKey binding shows "currentVk": 86 (V = VK code 86)
8. Restore the binding back to C (VK 67) and save

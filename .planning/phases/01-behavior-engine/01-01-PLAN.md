**BehaviorType enum:**
```cpp
enum class BehaviorType : uint8_t {
    HoldToToggle,  // hold inputVk -> hold outputVk continuously
    EdgeTrigger,   // rising edge on inputVk -> pulse outputVk for durationMs
    LongPress,     // tap -> pulse shortOutputVk; hold >= thresholdMs -> pulse longOutputVk
    WheelToKey,    // rising edge on inputVk -> send mouse wheel delta (wheelDelta)
};
```

**BehaviorDescriptor struct:**
```cpp
struct BehaviorDescriptor {
    BehaviorType type         = BehaviorType::HoldToToggle;
    WORD         outputVk     = 0;      // HoldToToggle, EdgeTrigger, LongPress short output
    WORD         longOutputVk = 0;      // LongPress only: output on sustained hold
    int          thresholdMs  = 400;    // LongPress: hold threshold in milliseconds
    int          durationMs   = 50;     // EdgeTrigger: output pulse duration in milliseconds
    DWORD        wheelDelta   = 120;    // WheelToKey: MOUSEEVENTF_WHEEL mouseData (120 = up, (DWORD)-120 = down)
};
```

**Per-binding state structs (one per behavior type, used as a union/array in the thread):**

**KeyTracker class:**
Track every key injected by the engine so we can release all of them during teardown. The set stays small (MAX_BINDINGS entries max).
```cpp
class KeyTracker {
public:
    void press(WORD vk)   { if (vk) active_.insert(vk); }
    void release(WORD vk) { active_.erase(vk); }
    void releaseAll() {
        for (WORD vk : active_) ReleaseKey(vk);
        active_.clear();
    }
private:
    std::set<WORD> active_;
};
```
Note: `ReleaseKey` is a forward-declared inline defined in CustomControlZ.cpp. The forward declaration at the top of the file is: `inline void ReleaseKey(WORD vk);`

**GenericLogicThreadFn declaration and implementation:**

The function signature matches LogicThreadFn:
```cpp
void GenericLogicThreadFn(GameProfile* profile, std::atomic<bool>& running);
```

Implementation logic:
1. Create `BindingState state[MAX_BINDINGS]` on the stack (zero-initialized via union constructor).
2. Create `KeyTracker tracker` on the stack.
3. Create `bool lastGameState = false`.
4. Loop while `running`:
   a. If `g_waitingForBindID != 0`: Sleep(50ms), continue.
   b. Check `IsGameRunning(profile)`. On state change:
      - If game stopped (was running, now not): call `tracker.releaseAll()`, call `SetTrayIconState(false, profile)`.
      - If game started (was stopped, now running): call `SetTrayIconState(true, profile)`.
      - Update `lastGameState`.
   c. If `!currentGameState`: Sleep(1000ms), continue.
   d. Snapshot binding VKs under `g_configMutex` into a local array `WORD localVk[MAX_BINDINGS]`.
   e. For each binding `i` in `[0, profile->bindingCount)`:
      - Dispatch to behavior handler by `profile->bindings[i].behavior.type`:
        - **HoldToToggle**: If `IsKeyDown(localVk[i])` and not `state[i].holdToggle.active`: `PressKey(desc.outputVk)`, `tracker.press(desc.outputVk)`, set active=true. If NOT `IsKeyDown(localVk[i])` and `state[i].holdToggle.active`: `ReleaseKey(desc.outputVk)`, `tracker.release(desc.outputVk)`, set active=false.
        - **EdgeTrigger**: On rising edge (key down, not fired): `PressKey(desc.outputVk)`, `tracker.press(desc.outputVk)`, `Sleep(desc.durationMs)`, `ReleaseKey(desc.outputVk)`, `tracker.release(desc.outputVk)`, set fired=true. On key up: set fired=false.
        - **LongPress**: On rising edge: record pressTime, set keyDown=true, longFired=false. While held and not longFired: if elapsed >= thresholdMs: `PressKey(desc.longOutputVk)`, `tracker.press(desc.longOutputVk)`, `Sleep(30)`, `ReleaseKey(desc.longOutputVk)`, `tracker.release(desc.longOutputVk)`, set longFired=true. On falling edge (key was down, now up): if not longFired: `PressKey(desc.outputVk)`, `tracker.press(desc.outputVk)`, `Sleep(30)`, `ReleaseKey(desc.outputVk)`, `tracker.release(desc.outputVk)`. Reset keyDown=false, longFired=false.
        - **WheelToKey**: On rising edge (key down, not pressed): send mouse wheel INPUT with `desc.wheelDelta`, set pressed=true. On key up: set pressed=false.
   f. `Sleep(10)`.
5. After loop exits (`running` became false): call `tracker.releaseAll()`.

WheelToKey injection (do not use PressKey/ReleaseKey — it's a mouse event):

INPUT inp = {};
